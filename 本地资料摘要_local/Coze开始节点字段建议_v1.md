# Coze 开始节点字段建议 v1

## 为什么要改字段

你之前的内容工作流更像通用内容助手。

现在既然已经进入奥斯翰招生场景，开始节点字段就应该贴业务，而不是继续用过于泛的输入。

## 第一版推荐字段

### 1. `school_topic`

你这条内容最核心的主题。

示例：

- `奥斯翰国际部四大优势`
- `AP课程适合什么样的学生`
- `港澳台联考和DSE的区别`
- `为什么国际路线更需要小班管理`

### 2. `program_path`

对应的课程方向。

示例：

- `学校总览`
- `OSSD`
- `AP`
- `A-Level`
- `日本`
- `韩国`
- `新加坡IFD`
- `港澳台联考`
- `DSE`

### 3. `target_audience`

内容面向谁。

示例：

- `初三家长`
- `正在考虑国际路线的家庭`
- `港澳台身份家庭`
- `想申请香港本科的学生和家长`

### 4. `material_summary`

把你整理好的资料摘要粘进来。

这里不要塞整份 PPT。

只放：

- 核心卖点
- 适合人群
- 升学路径
- 管理或师资优势

### 5. `content_type`

这条内容打算做成什么形式。

示例：

- `短视频口播`
- `短视频录屏讲解`
- `宣讲视频`
- `FAQ问答视频`
- `数字人口播`

### 6. `constraints`

告诉模型哪些约束必须遵守。

示例：

- `不能夸大升学结果`
- `表达必须准确`
- `适合抖音`
- `先讲结果再讲流程`
- `面向家长能听懂`

## 你下一版 Coze 最小输入模板

```text
school_topic：AP课程适合什么样的学生
program_path：AP
target_audience：正在考虑国际路线、但不了解AP的初三家长
material_summary：奥斯翰AP课程为College Board授权课程，授权代码579073，适合目标美国、香港及多国申请的学生，可用于展示高挑战学科能力，支持SAT和语言考试共同规划。
content_type：短视频口播
constraints：不能夸大，语言直接，先讲家长最关心的问题，适合抖音
```

1. `奥斯翰国际部四大优势`

3. `AP课程适合什么样的学生`
4. `A-Level课程适合什么样的学生`
5. `港澳台联考到底是什么`
6. `DSE为什么对内地学生更友好`
