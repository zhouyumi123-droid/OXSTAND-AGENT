# 系统模块规划

## 模块 1：知识资料模块

作用：

- 管理学校资料
- 区分最终口径和参考资料
- 给内容生成提供可信输入

当前状态：

- 已经开始建立资料分层
- 已有第一批结构化摘要

## 模块 2：内容生成模块

作用：

- 选题生成
- 最佳选题筛选
- 脚本生成
- 发布包装

当前状态：

- Coze 工作流已基本跑通
- 目前使用 DeepSeek-V3.2 作为默认主模型

## 模块 3：审核与校对模块

作用：

- 检查是否夸大
- 检查是否空话
- 检查是否符合学校口径
- 检查是否适合对应平台

当前状态：

- 还没有独立做成节点
- 这是下一个必须补的模块

## 模块 4：视频生产模块

作用：

- 根据脚本生成镜头清单
- 根据素材池生成剪辑投喂
- 为视频编辑工具输出固定风格指令

未来可能输出：

- 分镜脚本
- 口播镜头顺序
- 配图/素材建议
- 剪辑说明

## 模块 5：封面与视觉模块

作用：

- 自动生成封面标题
- 自动生成封面设计 brief
- 未来可接图片生成或模板渲染

## 模块 6：数字人模块

作用：

- 生成适合数字人的短句脚本
- 控制句长、停顿、语气
- 输出给数字人平台的专用文本

## 模块 7：多平台发布模块

作用：

- 生成不同平台版本
- 输出发布文案
- 对接自动发布工具

## 模块 8：软件工作台模块

作用：

- 把前面各模块收进一个操作界面
- 支持手动选择课程方向、目标人群、内容类型
- 可查看生成结果、修改结果、归档结果

这个模块未来可以做成：

- HTML 页面
- Web 应用
- 本地工作台
